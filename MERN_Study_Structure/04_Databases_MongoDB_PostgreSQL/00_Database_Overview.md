# Databases Overview (MongoDB & PostgreSQL)

## 📚 Core Topics Covered

1. **MongoDB (NoSQL)**
   - Document-oriented storage (Collections & Documents).
   - CRUD Operations & Mongoose ODM.
   - Indexing strategies (Single, Compound, TTL).
   - Aggregation Pipeline ($match, $group, $lookup).
   - Schema Design: Embedding vs. Referencing.

2. **PostgreSQL (Relational)**
   - SQL Fundamentals (SELECT, INSERT, UPDATE, DELETE).
   - Relational Algebra (Joins: Inner, Left, Right, Full).
   - Constraints: Primary Key, Foreign Key, Unique, Check.
   - Advanced SQL: CTEs, Window Functions, and Subqueries.
   - Transactions & ACID properties.

3. **Performance & Advanced Topics**
   - Query Execution Plans (EXPLAIN ANALYZE).
   - Database Normalization (1NF, 2NF, 3NF).
   - Stored Procedures, Functions, and Triggers.
   - Full-Text Search and JSONB handling.
   - Database Sharding & Replication basics.
