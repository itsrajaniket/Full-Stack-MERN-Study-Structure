# Backend Development Overview (Node.js, Express, NestJS)
> ✍️ **Author:** [Aniket Raj](https://github.com/itsrajaniket) | 📅 **Updated:** April 2025
---

## 📚 Core Topics Covered

1. **Node.js Fundamentals**
   - V8 Engine, Event Loop, Libuv.
   - Modules (CJS vs ESM), `fs`, `path`, `http` modules.
   - Asynchronous patterns (Promises, Async/Await).
   - Streams & Buffers for performance.

2. **Express.js**
   - Middleware architecture (Built-in, Custom, 3rd party).
   - Routing, Controllers, and Request/Response handling.
   - RESTful API design & CRUD operations.
   - Error handling & HTTP status codes.

3. **NestJS (Enterprise Architecture)**
   - Opinionated structure (Modules, Controllers, Providers).
   - Dependency Injection (DI) & Inversion of Control (IoC).
   - DTOs, Validation Pipes, Guards, and Interceptors.
   - Request Lifecycle deep-dive.

4. **Authentication & Authorization**
   - JWT (jsonwebtoken) & Passport.js strategies.
   - Session vs Token-based auth.
   - Role-Based Access Control (RBAC).
   - Password hashing with Bcrypt.

5. **Advanced Backend Patterns**
   - File Uploads with Multer & Cloudinary/S3.
   - Rate Limiting, Helmet, and CORS for security.
   - Real-time communication with WebSockets (Socket.io).
   - Caching with Redis for high-speed data retrieval.
   - Task Scheduling (Cron Jobs).
   - Microservices architecture basics in NestJS.

6. **Database & ORM Integration**
   - TypeORM & Prisma with NestJS.
   - Repository Pattern for data abstraction.
   - Database Migrations for version control.

## 🎓 Quick Interview Questions

- **Node.js**: Is Node.js single-threaded? Explain the Thread Pool.
- **Event Loop**: What are the phases of the Event Loop?
- **Middleware**: What happens if you forget to call `next()` in Express?
- **NestJS**: Why use NestJS over plain Express? What is DI?
- **JWT**: Where should you store a JWT on the client side for maximum security?
- **OAuth**: Explain the Authorization Code flow in Google Login.
- **Streams**: When would you use a Stream instead of `fs.readFile()`?
- **WebSockets**: How does a WebSocket handshake differ from a standard HTTP request?
- **Microservices**: What are the benefits of Microservices over Monoliths?
- **ORM**: Explain the N+1 query problem and how to solve it.
